# **App Name**: InfinityTools

## Core Features:

- Omni-Search Dashboard: A high-performance command palette and category grid to instantly find and launch any of the 100+ utilities.
- AI Productivity Engine: Generative AI tool for drafting ATS-optimized resumes, cover letters, and intelligent text summarization.
- Browser-Based File Processor: Local file processing for PDF merging, image compression, and format conversion without uploading sensitive data to a server.
- Real-time Resume Builder: Live preview markdown-style editor with custom templates and direct-to-PDF export functionality.
- User Tool Persistence: Secure storage to save resume drafts, tool configurations, and 'favorite' lists using Firestore.
- Developer Logic Suite: Functional utilities including JSON formatters, Base64 encoders, and Regex testers for rapid development tasks.
- Auth-Sync System: Authentication integration allowing users to access their saved projects and preferences across any device.

## Style Guidelines:

- Primary color: Vibrant Indigo (#6259ED) to evoke professionalism and technological sophistication.
- Background color: Deep Charcoal-Blue (#100F15) used for a sleek, immersive dark mode experience.
- Accent color: Sky Blue (#5E9EFF) to provide high-contrast visual cues for interactive elements.
- Headline font: 'Space Grotesk' for a technical, modern SaaS aesthetic; Body font: 'Inter' for maximum readability in data-heavy tools.
- Sharp, 2px stroke weight duo-tone icons from a modern library like Lucide to ensure a premium toolset feel.
- Modern glassmorphism effect using frosted background blurs (backdrop-filter) and subtle 1px border strokes to define card structures.
- Micro-interactions using Framer Motion for smooth tool transitions and subtle scaling effects on hover.
